    # Animated Profile Card

    # Descriptions

    Your task is to create a beautiful and engaging profile card to showcase your skills and talents and is a great way to advertise yourself.

    Your profile card should include but is not limited to:

    • Profile Image
    • Name
    • Contact Information
    • Short Descriptive Biography
    • Links to your social profiles
    • Something unique to you

    # Requirements:

    • Must include a minimum of three distinct animations.
    • Use a combination of opacity, transforms, rotations, scales or more to give your card an engaging feel.
    • Using CSS Pseudo Selectors or JavaScript click events to set up and toggle animations on click or hover.
    • Use of CSS and HTML to create and style your profile card.
    • Use of Animation Timing Delays for animation sequences.
    • Include a loading animation
    • Include interactive animations

    The midterm project will be completely created using Brackets or your text editor of chose. While you can use Codepen to test your ideas, you will need to build everything on your computer using HTML, CSS, and JS files.
