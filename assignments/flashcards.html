    # Flashcards

    Objective

    For this assignment you will be demonstrating your understanding of JavaScript and CSS to create a flashcard app. You will need to create elements and event listeners using JavaScripts and design the flashcards using CSS.

    Game Play

    The player will be presented with a different decks of flashcards. Click on a deck will start a quiz. A quiz will present each card in the deck one at a time with the answer hidden. The player can the view the answer by clicking the show answer button. The player can move through the deck by using the next and previous buttons. At any time, the player can exit the quiz by clicking the "quit" button. 

    Requirements

    The following requirement must be met in order to complete the assignment successfully:

    1. There must be at least 2 decks and each deck must have at least 5 cards. Each card will consist of a question and an answer. The questions can be of any topic you wish, but the decks CANNOT share questions.
    2. All decks must be dynamically added to the page using JavaScript, when the page loads.
    3. Clicking on a deck will start a quiz of the cards from the selected deck.
    4. A quiz will display each card, one at a time, with the answer being hidden. 
    5. The following quiz controls should be made available:
      1. A control to show the answer to the current card
      2. A control to move to the next card. If at the last card, then go to the first card.
      3. A control to move to the previous card. If at the first card, then go to the last card.
      4. A control to quit the quiz
    6. When the a new card is displayed, the answer should be hidden, even if the answer was shown previously.
    7. At anytime the player can quit the quiz by using quit control.
    8. You may design your Flashcards app how you deem best, but it must be responsive and use the Flexbox model. The entire application must be on one HTML page. All changes to the page, starting quiz, displaying the cards, and showing and hiding of the answer must be done using JavaScript.

    Example

    Hints

    1. It will be easier to create a static version of the Flashcards app using HTML and CSS. Once you have finalized the HTML and CSS, start pulling out any elements that will need to be added or changed by JavaScript, while writing the JavaScript code to add them back in.
    2. Create an object to hold the decks and cards.
    3. Create an object to hold the quiz status. You will need to keep track of the current deck, current question, and deck total cards.
    4. Add an id attribute to any elements you need to retrieve using the getElementById() method.
    5. When hiding elements, it is better to use opacity: 0 and visibility: hidden then display: none.
