          # Dominoes

          # Objective

          For this assignment you will be demonstrating your skills of DOM manipulation with JavaScript and Flexbox, by randomly generating and displaying 100 dominoes and one the page.

          # Requirements

          The following requirement must be met in order to complete the assignment successfully:

          1. Create a folder with the name dominoes. In the folder, create 3 files: index.html, index.js, and style.css.
          2. The index.html file should have the following:
            1. Full HTML structure including <doctype>, <html>, and <body> tags
            2. The <title> tag should have the text: YOUR NAME's Dominoes.
            3. A <link> tag linking to style.css
            4. An <h1> tag with the text: YOUR NAME's Dominoes.
            5. A container for the dominoes (e.g. <div id="dominoes"></div>)
            6. A <script> tag linking to dominoes.js
          3. The style.css file should have the following:
            1. All of the styles for creating page and dominoes layouts
            2. Flexbox layout must be used for the page layout
            3. The dominoes must be laid out as to fill the page
            4. Flexbox layout must be used for each domino layout and the dots on the domino
            5. Each domino must have two dots sections with 0 to 6 dots
            6. Each domino must be have a width that is roughly 1/2 the height
            7. The dots on the domino must be laid out properly (see below). NOTE: A domino with five dots will require some extra styles.
            8. The domino, the dots, and the lines may be any color you wish
          4. The dominoes.js file should have the following:
            1. Create a function called createDot(). This function will create the HTML elements and attributes to create the HTML structure of a single dot. This function will be return the dot HTML element or string. The HTML element should NOT be added to the DOM.
            2. Create a function called createSection(). This function will will create the HTML elements and attributes to create the the HTML structure of a dots section of the domino. This will call createDot to get the dot elements. The number dots needed will be determined randomly. This function will return the dots HTML element or string. The HTML element should NOT be added to the DOM.
            3. Create a function called createDomino(). This function will create the HTML elements and attributes to create the HTML structure of a single domino element. This function will call the createDots function to create the top and bottom section of the domino. This function will return the domino HTML element or string. It should NOT be added to the DOM.
            4. Create a document fragment or an array that will serve as a virtual container to hold all 100 domino elements.
            5. Retrieve the dominoes container from the DOM using one of the HTML selector methods.
            6. Append the document fragment holding all 100 domino elements to the dominoes container.

          # Hints

          1. It has been mentioned that the 2 and 3 dots do NOT follow the traditional layouts of dominoes. I chose the layouts I did because I believed they required the least amount of customization. Once you have the flexbox layout set up correctly the dots fall into place, with the exception of 5. HOWEVER, if you want to create dots layouts that match the traditional domino layouts, you will NOT lose points for doing so. 
          2. To create the dominoes using flexbox, you may need to add more HTML structure than you might think. For example, you may need to create the dot inside a dot container OR use the ::before or ::after psuedo-classes. 
          3. You need use flex-basis in order to get the dots formatted correctly. You can learn more about flex-basis on MDN or CSS-Tricks.
          4. On Codepen, I created Flexbox Letters, which uses flexbox build letters from boxes. Hopefully, some more insight how on to use flexbox for layouts.
