    # Crazy Cats

    Objective

    For this assignment you will be demonstrating that you understand how to create and use JavaScript objects and arrays.

    Requirements

    To successfully complete the assignment, the following requirements must be met. 

    1. Create a variable with the name cat1 and assign it an object with the following information about a cat:  
      1. Name (String)
      2. Age (Number, in years)
    2. Send a message to the console using cat1 object. The messages should have the following pattern:
      1. [Name] is [Age] years old.
    3. Create a variable with the name cat2 and assign it an object with the following information about a new cat (should be different than cat1): 
      1. Name (String)
    4. Add the following information to the cat2 object:
      1. Species (String)
      2. Colors (String)
    5. Send a message to the console using cat2 object. The message should have the following pattern:
      1. [Name] is a [Species] cat with [Colors] fur.
    6. Create a variable with the name cat3 and assign it an object with the following information about a new cat (should be different than cat1 and cat2)
      1. Name (String)
    7. Add the following information to the cat3 object:
      1. Likes (Array, minimum of 4)
    8. Using an array method, remove the last item from the Likes array.
    9. Using an array method, sort the Likes array alphabetically.
    10. Send a message to the console using cat3 object. The message should have the following pattern:
      1. [Name] likes [Like 1], [Like 2], and [Like 3]
    11. Create a variable with the name cat4 and assign it an object with the following information about a new cat (should be different than cat1, cat2 and cat3)
      1. Name (String)
      2. Age (Number, in years)
      3. Species (String)
      4. Colors (String)
    12. Add the following information to the cat4 object:
      1. Dislikes (Array, minimum of 3)
    13. Using an array methods, sort the Dislikes in reverse alphabetical order
    14. Remove the properties Age from the cat4 object
    15. Send a message to the console using cat4 object. The message should have the following pattern:
      1. [Name] is a [Colors] [Species] cat who doesn’t likes [Dislike 1], [Dislike 2], and [Dislike 3]
