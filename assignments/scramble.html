              # Scramble

              # Objective

              For the midterm project you will be demonstrating your understanding JavaScript to create a Scramble game.

              # Game Play

              When the page loads, the player will be presented with a list of instructions in the browser's console. The player can request these instructions again at any time by using the help() command.

              After the player starts the game, they will be given a scrambled word. The player will be able to guess the correct spelling of the word using the guess() command. If correctly guessed the word the player will receive a point. If guessed incorrectly, they player will receive a strike and a new scrambled will appear. The player will only be able to guess once per word.

              The player can also use the pass() command. With the pass() command a player can "pass" on a word. The word will be randomly added back to the list. The player can only pass a limited number of times.

              The game ends when all words in the list have been guessed or after the player has received the maximum number of strikes.

              After the games ends, the player will be present with a total number points earned for that game.

              # Requirements

              The following requirement must be met in order to complete the assignment successfully:

              1. Create a words array to hold the words that will be used in the game. To keep things simple do NOT include words with spaces or special characters. The array should include at least 30 words.
              2. Create a game object. The game object will hold the current game's status including:
                1. The game active status
                2. The current word
                3. The number of strikes
                4. The number of points
                5. The maximum number of allowed strikes
              3. Create a help() function that will return a set of instruction on how to play Scramble must appear in the console. The instruction should include a list of commands or functions the player can use during the game.
              4. When the page loads, call the help() function show that the instructions are visible in the console.
              5. Create a start() function, which is used to start a new game, will do the following:
                1. Reset all the game status properties
                2. Make a copy of the words array and store it in the game object
                3. Shuffle the words array copy (see function below).
                4. Select a word from the shuffled array save to the game object.
                5. Display a scrambled version of the word to the player
              6. If a game currently active than a new game CANNOT be started.
              7. Create a guess() function that will be used by the player to guess the word and will do the following:
                1. Check to see if the guessed word matches the current word unscrambled. Case should NOT matter.
                2. If the words do match the player will receive a point and the next word should be displayed
                3. If the words do NOT match the player will receive a strike and the next word should be displayed (A player can only make one guess per word)
              8. The player should NOT be able to make guesses if there is no active game.
              9. Create a pass() function that will be used by the player to pass on a word and will do the following:
                1. Check if player has any passes left
                2. Add the current word in a random spot in the list
                3. Display the next word
              10. The game will end if there are no more words in the list (Each word can only receive one guess) OR the player has received the maximum number of strikes
              11. When the game ends the player total points should be displayed.
              12. After the game ends the player should be able to start a new game using the start() function

              # Shuffle Function

              The following shuffle() function can be used for this project.

              /**
 * shuffle()
 * Shuffle the contents of an array
 *   depending the datatype of the source
 * Makes a copy. Does NOT shuffle the original.
 * Based on Steve Griffith's array shuffle prototype
 * @Parameters: Array or string
 * @Return: Array - a shuffled version of the array
 */
function shuffle (src) {
  const copy = [...src]

  const length = copy.length
  for (let i = 0; i < length; i++) {
    const x = copy[i]
    const y = Math.floor(Math.random() * length)
    const z = copy[y]
    copy[i] = z
    copy[y] = x
  }

  return copy
}
